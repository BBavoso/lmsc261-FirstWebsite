Introduction to Game Development

Here are a few of the different parts of game developement. This is not a comlete list, but just a few of the areas to get started.

Different parts of game development:

Programming

Game development uses several different methods to program logic into the game. The most popular languages for game development are c++, which you will commonly see with the Unreal game engine, and c#, which you will see with the Unity engine.

Design

Game Design is a very important and very broad part of game developement. Over the course of developement of a game you will have to look at what you have so far and really study how your game systems interact and how the player interacts with them. Game designers are responsible for making sure that a game is actually fun to play.

Art

Art is all of the visuals that you will see in games. Game art is mainly divided into 2D and 3D

2D

2D art is all of the art in the game that is in 2D. For a 2D game this is all of the assets, but for a 3D game this would include parts of the game like the menu and the HUD.

Some common 2D art programms include:

  • Aseprite - Pixel Art
  • Pixquare - Pixel Art
  • Pixel Studio Pro - Pixel Art on Android
  • Krita - Painting / Drawing
  • Procreate - Painting / Drawing on IPad
  • Photoshop - Photo Editing
  • Paint.Net - Open Source Photo Editing

3D

3D art is all of the 3D assets in a game. This includes things like character models and the game world itself.

  • 3ds Max - modeling, architectural visualization, and game design
  • Maya - modeling, animation
  • Houdini - procedural modeling, complex simulations, and visual effects
  • Blender - free and open source modeling, animation

Sound

Sound includes the music and the sound effects of the game.

  • Ableton - DAW
  • Reaper - DAW
  • Pro Tools - DAW
  • Wwise - Audio Middleware
  • fmod - Audio Middleware
